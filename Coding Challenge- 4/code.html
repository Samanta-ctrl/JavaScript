JavaScript Coding Challenge 4

Lets remember the first coding challenge where Mark and John compared their BMIs. Let's now implements the same functionality with objects and methods.

1. For each of them, create an object with properties for their full name,mass and height.

2. Then, add a method to eah objects to calculate the BMI. save the BMI to the objects and also return it from the method.

3. In the end, log to the console who has the highest BMI, together with the full name and the respective BMI. Don't forget they might have the same BMI.


Remember: BMI = mass / height ^ 2 = mass / (height * height). Mass in kg and height in meter.
